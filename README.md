> [FastNoise2 by Auburn](https://github.com/Auburn/FastNoise2) packaged in UPM

Platform support:

* Linux x64, Android x64
* macos Universal, macos Intel
* Windows x64

Feature support:

* `NativeTextureXD<T>`
* Burst/Jobs/IL2CPP compatible FastNoise2 bindings
* Noise Graphs Editor support
* Baked Noise Assets
